## 📊 What I Built

A complete ETL data platform with only **Python stdlib** - zero external dependencies!

### Tech Stack
- Python (stdlib only)
- SQLite
- Custom ETL Pipeline
- Web Dashboard

### Features
✅ ETL Pipeline: Generate → Ingest → Clean → Aggregate → Load
✅ SQLite Database with star schema
✅ Web Dashboard http://localhost:8888
✅ One-click CSV Export
✅ No pip install needed!

---

## 🚀 How to Run

```bash
./start.sh
```

That's it! Then open http://localhost:8888

---

## 📁 Project Structure

```
Data Platform/
├── start.sh              # Main: run both
├── run_pipeline.sh      # ETL
├── ingestion/          # Data ingestion
├── processing/         # ETL transforms
├── warehouse/          # SQLite DB
└── dashboard/         # Web UI
```
